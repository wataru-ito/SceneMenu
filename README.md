SceneMenu
================

Abstract
--------

作業シーンの移動をアシストしてくれるエディタ拡張

Menuから楽々移動

![image](https://github.com/wataru-ito/SceneMenu/blob/master/scenemenu_image.png)




Usage
--------

* 「Menu > Scene > 設定」

![Setting](https://github.com/wataru-ito/SceneMenu/blob/master/scenemenu_image2.jpg)

```
【限定】シーンの検索対象が限定される
【除外】そのフォルダを検索対象外とする
【階層】メニュー上で１階層下げる（まとめる）
```
